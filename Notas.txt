Para generar el contenido estático se utiliza:
pelican content

Para exportar para gh-pages se utiliza:
ghp-import output

Para iniciar el servidor:
python -m pelican.server

Hay que clonar el repositorio que contiene los temas de pelican antes.
git clone --recursive https://github.com/getpelican/pelican-plugins

Para arreglar el problema con graphviz
	Descargar https://graphviz.gitlab.io/_pages/Download/Download_windows.html
	pip install graphviz
